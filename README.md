# telegram-movie-predictor 🎬

Telegram-бот для подбора фильмов из топ-250 КиноПоиска. Бот умеет искать фильмы по жанру, десятилетию, стране, рейтингу, комбинации этих параметров, а также **подбирать фильм по настроению** — пользователь описывает своими словами, что хочет почувствовать («хочется поплакать», «нужно что-то страшное», «хочу романтики»), а бот с помощью лингвистического анализа текста определяет подходящие жанры.

## Возможности

- 🎭 Подбор по настроению (свободный текстовый запрос)
- 🎞️ Поиск по жанру
- 🗓️ Поиск по десятилетию
- 🌏 Поиск по стране
- ⭐ Поиск по рейтингу (7.0+ / 8.0+ / 9.0+)
- 🧩 Комбинированный поиск (жанр + десятилетие + страна)
- 🎲 Случайные фильмы

## Данные

Используется датасет топ-250 фильмов КиноПоиска:

- Источник: [Kinopoisk Top-250 Russian Dataset на Kaggle](https://www.kaggle.com/datasets/alexandertesemnikov/kinopoisktop250russiandataset)
- Файл в проекте: `kinopoisk_top250_cleaned.csv` (предобработанная версия)
- Поля: `rating`, `movie`, `year`, `country`, `rating_ball`, `overview`, `director`, `screenwriter`, `actors`, `url_logo`, `Genres_normalized`
- 250 фильмов, 22 нормализованных жанра (драма, комедия, мелодрама, ужасы, триллер, боевик, фантастика, фэнтези, детектив, криминал, биография, история, военный, спорт, семейный, мультфильм, аниме, приключения, музыка, мюзикл, вестерн, фильм-нуар)

## Структура проекта

| Файл | Описание |
|---|---|
| `main.py` | Инициализация бота и диспетчера, загрузка CSV, функции фильтрации фильмов, клавиатуры, все обработчики команд/сообщений/callback-ов |
| `emotion_analyzer.py` | Лингвистический модуль. Лемматизирует текст пользователя (pymorphy3) и сопоставляет его со словарём эмоциональных категорий, возвращая категорию настроения и список подходящих жанров |
| `kinopoisk_top250_cleaned.csv` | Датасет фильмов |
| `requirements.txt` | Зависимости проекта |
| `technical_documentation.docx` | Техническая документация: архитектура бота, описание модулей и логики работы |
| `user_guide.docx` | Руководство пользователя: как пользоваться ботом, описание всех команд и кнопок меню |

## Установка

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/p0ggerz/telegram-movie-predictor.git
   cd telegram-movie-predictor
   ```

2. Создать и активировать виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate   # Linux/macOS
   venv\Scripts\activate      # Windows
   ```

3. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   pip install pymorphy3-dicts-ru
   ```

4. Указать токен Telegram-бота, полученный у [@BotFather](https://t.me/BotFather), в переменной `BOT_TOKEN` в `main.py`.

## Запуск

```bash
python main.py
```

После запуска бот доступен в Telegram. Команды `/start` и `/help` выводят главное меню с кнопками всех режимов поиска.
